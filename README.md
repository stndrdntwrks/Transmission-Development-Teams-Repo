

# 문서 태그(Tag)

> 태그(Tag)란 ? `#키워드` 형식으로 작성하여 Vault 내에서 빠른 검색을 위해서 사용

| 키워드          | 의미              |
| ------------ | --------------- |
| `이슈`         | 발생한 이슈 문서 태그    |
| `이슈해결`       | 해결한 이슈 문서 태그    |
| `yyyy-MM-dd` | 날짜 태그           |
| `회의록`        | 전송개발팀 회의록 문서 태그 |
| `Documents`  | 문서 모음 태그        |
| `TODO`       | 추후 진행 업무 태그     |
| `MCMP`       | MCMP 플랫폼 문서 태그  |
| `${각 프로세스명}` | 각 프로세스 별 문서 태그  |
